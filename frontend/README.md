# Frontend - Face Recognition

This folder contains the frontend interface for the face recognition system.

## Structure

- `index.html` - Main HTML page
- `style.css` - Styling
- `script.js` - JavaScript functionality

## Getting Started

Open `index.html` in your browser to view the frontend.
